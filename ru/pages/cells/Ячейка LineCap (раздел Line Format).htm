Ячейка LineCap (раздел Line Format)

Указывает тип наконечника линии (круглый, квадратный или удлиненный).

Значение

Стиль конца линии

0

Круглый

1

Квадратный

2

Удлиненный

Заметки

Задать значение этой ячейки также можно в диалоговом окне Линия (чтобы вызвать его, на вкладке Главная в группе Фигура щелкните элемент Линия, наведите указатель на пункт Стрелки и выберите команду Другие стрелки).

Чтобы получить ссылку на ячейку LineCap по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

LineCap

Чтобы получить ссылку на ячейку LineCap по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionObject

Индекс строки:

visRowLine

Индекс ячейки:

visLineEndCap
