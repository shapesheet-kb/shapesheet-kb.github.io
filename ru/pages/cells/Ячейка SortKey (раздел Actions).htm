Ячейка SortKey (раздел "Actions")

Число, определяющее порядок действий в контекстном меню или меню тегов действий.

  Примечание.

В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами.

Заметки

Действия, назначенные тегам действий или контекстным меню, отображаются в порядке возрастания назначенных им чисел. Если у двух строк действий одинаковое значение ячейки SortKey, порядок соответствует порядку самих строк. По умолчанию используется значение 0 (нуль).

Чтобы создать ссылку на ячейку SortKey по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Actions.имя.SortKey,

где Actions.имя — это имя строки действий

Чтобы создать ссылку на ячейку SortKey из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionAction

Индекс строки:

visRowAction + i,

где i = 0, 1, 2...

Индекс ячейки:

visActionSortKey
