Ячейка Scale (раздел "Character")

Регулирует ширину шрифта. По умолчанию имеет значение 100 %.

Заметки

Чтобы уменьшить ширину шрифта, задайте значение от 1 до 99 %, а чтобы увеличить ее — значение между 101 и 600 %.

Значение этой ячейки можно также задать с помощью диалогового окна Текст (на вкладке Главная щелкните стрелку Шрифт).

Чтобы создать ссылку на ячейку Scale по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Char.FontScale[i], где i = <1>, 2, 3...

Чтобы создать ссылку на ячейку Scale из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionCharacter

Индекс строки:

visRowCharacter + i, где i = 0, 1, 2...

Индекс ячейки:

visCharacterFontScale
