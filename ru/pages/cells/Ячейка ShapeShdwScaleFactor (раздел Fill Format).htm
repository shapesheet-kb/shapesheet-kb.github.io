Ячейка ShapeShdwScaleFactor (раздел "Fill Format")

Определяет процент, на который можно увеличить или уменьшить тень фигуры.

Заметки

У каждой тени есть затененный центр вращения, то есть точка, соответствующая центру вращения фигуры. Например, если центр вращения фигуры находится в центре фигуры, затененный центр вращения будет точкой в центре тени. При применении масштаба к простым теням увеличение выполняется относительно затененного центра вращения. В случае с тенью, расположенной с наклоном, увеличение применяется к наклону.

Чтобы задать это значение для всех фигур на странице, используйте ячейку ShdwScaleFactor в разделе "Page Properties".

Это значение соответствует значению параметра Увеличение диалогового окна Тень (на вкладке Главная в группе Фигура выберите пункт Тень и командуПараметры тени).

Чтобы создать ссылку на ячейку ShapeShdwScaleFactor по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShapeShdwScaleFactor

Чтобы создать ссылку на ячейку ShapeShdwScaleFactor из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowFill

Индекс ячейки:

visFillShdwScaleFactor
