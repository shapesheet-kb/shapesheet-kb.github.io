Ячейка X (раздел "Annotation")

Координата x знака примечания в координатах страницы.

Примечания

Чтобы создать ссылку на ячейку X по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Annotation.X[i]
, где i = <1>, 2, 3...

Чтобы создать ссылку на ячейку X из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionAnnotation

Индекс строки:

visRowAnnotation + i,
где i = 0, 1, 2...

Индекс ячейки:

visAnnotationX
