Ячейка ShdwBkgnd (раздел "Fill Format")

Определяет цвет, используемый для фоновой заливки узора заливки тени фигуры.

Заметки

Чтобы задать цвет, введите число от 0 до 23 (индекс из коллекции цветов).

Чтобы ввести пользовательский цвет, используйте функцию RGB или HSL. Значением пользовательского цвета является цвет RGB, и в окне таблицы свойств фигуры будет отображаться RGB(r, g, b), а не число. При использовании в числовых операциях пользовательским цветам назначаются значения 24 и выше.

Прозрачность цвета фона узора заливки тени фигуры можно задать с помощью ячейки ShdwBkgndTrans.

Чтобы создать ссылку на ячейку ShdwBkgnd по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

ShdwBkgnd

Чтобы создать ссылку на ячейку ShdwBkgnd из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowFill

Индекс ячейки:

visFillShdwBkgnd
