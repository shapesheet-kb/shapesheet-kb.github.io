Ячейка Invisible (раздел Actions)

Указывает, отображается ли действие в меню тегов действий или в контекстном меню.

  Примечание.

В предыдущих версиях Microsoft Visio теги действий назывались смарт-тегами.

Значение

Описание

TRUE

Действие не отображается в меню.

FALSE

Действие отображается в меню (вариант по умолчанию).

Заметки

Чтобы получить ссылку на ячейку Invisible по имени из другой формулы или из программы с помощью свойства CellsU, используйте указанные ниже параметры.

Имя ячейки:

Actions.имя.Invisible,

где Actions.имя — имя строки действий

Чтобы получить ссылку на ячейку Invisible по индексу из программы, воспользуйтесь свойством CellsSRC с указанными ниже аргументами.

Индекс раздела:

visSectionAction

Индекс строки:

visRowAction + i,

где i = 0, 1, 2...

Индекс ячейки:

visActionInvisible
