Ячейка SortKey (раздел "Shape Data")

Возвращает строку, которая влияет на порядок, в котором элементы выводятся в окне Данные фигуры.

Заметки

Вычисления, которые используются для сравнения значений SortKey, зависят от языкового стандарта и не учитывают регистр. Если значения SortKey совпадают, данные фигуры выводятся в порядке, соответствующем порядку строк. Данные фигуры, не имеющие ключа сортировки, выводятся после данных с ключом сортировки.

Ниже приведен пример использования клавиш сортировки для вывода данных фигуры в окне Данные фигуры в следующем порядке: "Номер элемента", "Количество", "Цена".

Строка, Подпись и SortKey относятся к ячейкам в строке данных фигуры. В этом случае строки данных фигуры имеют имена.

Строка

Подпись

SortKey

Prop.Item

Номер элемента

1

Prop.Price

Цена

3

Prop.Quan

Количество

2

Чтобы создать ссылку на ячейку SortKey по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Prop.имя.SortKey,

где Prop.имя — это имя пользовательской строки свойств

Чтобы создать ссылку на ячейку SortKey из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionProp

Индекс строки:

visRowProp + i,

где i = 0, 1, 2...

Индекс ячейки:

visCustPropsSortKey
