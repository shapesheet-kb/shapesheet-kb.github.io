Ячейка Width (раздел "Shape Transform")

Содержит значение ширины указанной фигуры в единицах документа. По умолчанию для расчета ширины одномерной фигуры используется следующая формула:

= SQRT((EndX - BeginX) ^ 2 + (EndY - BeginY) ^ 2)

Примечания

Чтобы создать ссылку на ячейку Width по ее имени из другой формулы или из программы с помощью свойства CellsU, используйте:

Имя ячейки:

Width

Чтобы создать ссылку на ячейку Width из программы по индексу, используйте свойство CellsSRC со следующими аргументами:

Индекс раздела:

visSectionObject

Индекс строки:

visRowXFormOut

Индекс ячейки:

visXFormWidth
