Раздел Connection Points

Содержит строку и ячейки для каждой из точек соединения, указанных для фигуры.

Заметки

Точки соединения можно задать в разделе Connection Points окна таблицы свойств фигуры или с помощью функции Точки соединения на вкладке Главная.

Если у фигуры нет раздела Connection Points, его можно вставить в окне таблицы свойств фигуры. Для этого щелкните это окно правой кнопкой мыши и выберите команду Вставить раздел, а затем — Точки соединения. Чтобы добавить точки соединения в имеющийся раздел Connection Points, щелкните строку правой кнопкой мыши и в контекстном меню выберите команду Вставить строку.

Чтобы отобразить дополнительные ячейки, которые можно использовать для временных вычислений, измените тип строки для точки соединения в этом разделе. Для этого щелкните строку правой кнопкой мыши и в контекстном меню выберите команду Изменить тип строки.
