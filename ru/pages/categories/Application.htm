LANGUAGE

Возвращает код языка, соответствующий аргументу, в виде самого кода языка и региональных параметров BCP47.

Синтаксис

LANGUAGE(lcid_or_bcp47)

VERSION

Возвращает целое число, совпадающее с порядковым номером версии активного приложения Microsoft Visio.

Синтаксис

VERSION()
