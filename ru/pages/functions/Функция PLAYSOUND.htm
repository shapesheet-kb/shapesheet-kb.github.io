Функция PLAYSOUND

Воспроизводит звуковой файл или системный звук.

Синтаксис

PLAYSOUND("filename"|"alias",isAlias,beep,synch)

Parameters

Имя

Требуется/Необязательно

Тип данных

Описание

filename

Требуется

Строка

Имя звукового файла, который необходимо воспроизвести.

alias

Требуется

Строка

Системный звук, представленный псевдонимом.

isAlias

Требуется

Логический

Указывает, является ли предшествующее выражение псевдонимом или именем файла; чтобы указать на псевдоним, используйте ненулевое значение.

beep

Требуется

Логический

Указывает, следует ли приложению Microsoft Visio выдать сигнал, если невозможно воспроизвести звук; для выдачи сигнала используйте ненулевое значение.

synch

Требуется

Логический

Указывает, должны ли звуки воспроизводиться асинхронно (0) или синхронно (1).

Заметки
Звуки обычно воспроизводятся асинхронно, чтобы можно было продолжать работу в приложении Visio во время воспроизведения. Чтобы связать несколько звуков, воспроизведите их синхронно, иначе некоторые из них могут не быть воспроизведены.

Пример 1

PLAYSOUND("chord.wav", 0, 0, 0)

Воспроизводит звуковой файл chord.wav асинхронно без предупреждающего сигнала.

Пример 2

PLAYSOUND("СистемныйСигнал", 1, 0, 0)

Воспроизводит системный сигнал асинхронно без предупреждающего сигнала.
