Функция RAND

Возвращает случайное число с плавающей запятой от 0,0 до 1,0. При каждом вычислении результата этой функции (один раз в минуту согласно системным часам) возвращается новое число.

Синтаксис

RAND( )

Возвращаемое значение
Число с плавающей запятой

Заметки
С помощью этой функции можно создавать эффекты анимации, задавая для свойств фигур случайные значения.

Пример

RAND( )

Возвращает десятичную дробь (например, 0,3503).
