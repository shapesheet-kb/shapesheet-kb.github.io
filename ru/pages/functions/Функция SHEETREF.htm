Функция SHEETREF

Возвращает маркер ссылки на лист для указанной фигуры. Эта функция используется в других функциях, которые ожидают маркер ссылки на лист.

Синтаксис

<sheetref>!SHEETREF()

Возвращаемое значение
Ссылка на ячейку

Примечания

Эту функцию можно использовать в других функциях, которые берут маркер ссылки на лист.
